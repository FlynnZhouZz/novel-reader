# 小说阅读器 - 需求文档

## 1. 项目概述

### 1.1 项目简介

小说阅读器是一个基于 B/S 架构的在线小说阅读平台，用户可以通过浏览器阅读小说作品。项目分为客户端（client）和服务端（server）两部分，客户端提供用户交互界面，服务端提供数据接口和数据存储。

### 1.2 技术栈

| 层级 | 技术选型 |
|------|----------|
| 客户端 | React、Next.js、Ant Design、Redux、Axios |
| 服务端 | Node.js（Express/Koa） |
| 数据库 | MySQL |

### 1.3 项目结构

```
novel-reader/
├── client/          # 客户端（Next.js 前端项目）
├── server/          # 服务端（Node.js 后端项目）
└── docs/            # 项目文档
```

## 2. 用户体系模块

### 2.1 注册

#### 2.1.1 邮箱 + 验证码注册

| 字段 | 说明 |
|------|------|
| 邮箱 | 必填，合法邮箱格式，不可重复注册 |
| 验证码 | 必填，6 位数字，发送至用户邮箱，有效期 5 分钟 |
| 密码 | 必填，8-20 位，需包含字母和数字 |
| 确认密码 | 必填，需与密码一致 |

**业务规则：**
- 邮箱已注册时提示"该邮箱已被注册"
- 验证码发送后 60 秒内不可重复发送
- 验证码错误时提示"验证码错误或已过期"
- 注册成功后自动登录

#### 2.1.2 邮箱 + 密码注册

| 字段 | 说明 |
|------|------|
| 邮箱 | 必填，合法邮箱格式，不可重复注册 |
| 密码 | 必填，8-20 位，需包含字母和数字 |
| 确认密码 | 必填，需与密码一致 |

**业务规则：**
- 邮箱已注册时提示"该邮箱已被注册"
- 注册成功后自动登录

## 3. 小说管理模块

### 3.1 小说信息

| 字段 | 说明 |
|------|------|
| 小说名称 | 必填，最多 100 个字符 |
| 封面 | 选填，支持 JPG/PNG/WEBP 格式，最大 2MB |
| 简介 | 选填，最多 2000 个字符 |
| 作者 | 必填，最多 50 个字符 |

### 3.2 小说目录

- 每部小说包含一个有序的章节目录
- 目录按章节顺序号排列
- 目录展示章节标题和章节序号

### 3.3 小说章节

| 字段 | 说明 |
|------|------|
| 章节标题 | 必填，最多 200 个字符 |
| 章节内容 | 必填，小说正文内容 |
| 章节序号 | 必填，用于排序 |
| 所属小说 | 必填，关联小说 ID |

### 3.4 小说上传（通过 crawler-novels 项目）

**业务规则：**
- 支持从 crawler-novels 项目采集的小说数据进行批量上传
- 上传时自动创建小说信息和章节目录
- 支持增量更新（已存在的章节不重复创建）
- 上传过程需展示进度

### 3.5 小说列表与搜索

**业务规则：**
- 首页展示小说列表，支持分页
- 支持按小说名称搜索
- 支持按作者搜索
- 小说列表展示：封面、名称、作者、简介（截断）

### 3.6 小说详情

- 展示小说完整信息（封面、名称、作者、简介）
- 展示章节目录列表
- 点击章节进入阅读器

## 4. 小说阅读器模块

### 4.1 核心阅读功能

- 展示当前章节的小说正文内容
- 支持上一章 / 下一章切换
- 页面顶部展示：小说名称、当前章节标题
- 页面底部展示：上一章 / 下一章按钮、目录入口

### 4.2 背景颜色设置

提供预设背景颜色方案，用户可自由切换：

| 模式 | 颜色方案 |
|------|----------|
| 默认（白色） | 背景 #FFFFFF，文字 #333333 |
| 护眼（绿色） | 背景 #C7EDCC，文字 #333333 |
| 牛皮纸 | 背景 #F5E6C8，文字 #333333 |
| 粉色 | 背景 #FFDCE5，文字 #333333 |
| 深色 | 背景 #1A1A1A，文字 #CCCCCC |

**业务规则：**
- 用户选择后持久化保存（localStorage）
- 切换后即时生效

### 4.3 字体大小设置

| 档位 | 字体大小 |
|------|----------|
| 小 | 14px |
| 默认 | 16px |
| 中 | 18px |
| 大 | 20px |
| 特大 | 24px |

**业务规则：**
- 用户选择后持久化保存（localStorage）
- 切换后即时生效，内容区域自适应重新排版

### 4.4 高亮 / 暗黑模式

| 模式 | 说明 |
|------|------|
| 高亮模式 | 明亮主题，默认白色背景 |
| 暗黑模式 | 暗色主题，深色背景浅色文字 |

**业务规则：**
- 切换后即时生效
- 与背景颜色设置联动（暗黑模式下使用深色背景方案）
- 持久化保存用户偏好

### 4.5 翻页模式

提供四种翻页模式供用户选择：

| 模式 | 说明 |
|------|------|
| 上下翻页 | 传统网页滚动方式，内容从上到下排列 |
| 平移翻页 | 左右滑动切换章节，页面整体平移过渡 |
| 覆盖翻页 | 新页面从右侧滑入覆盖旧页面 |
| 仿真翻页 | 模拟真实书本翻页效果，带有 3D 翻转动画 |

**业务规则：**
- 用户选择后持久化保存（localStorage）
- 切换后即时生效
- 平移翻页、覆盖翻页、仿真翻页支持手势滑动（触摸/鼠标拖拽）
- 翻页到边界时提示"已是第一章"或"已是最后一章"

### 4.6 自动阅读

| 设置项 | 说明 |
|--------|------|
| 开关 | 开启/关闭自动阅读 |
| 速度 | 可设置滚动速度（慢/中/快），或自定义速度值 |

**业务规则：**
- 开启后页面自动匀速向下滚动
- 滚动到底部时自动切换至下一章
- 用户手动滚动或点击屏幕时暂停自动阅读
- 速度设置持久化保存

### 4.7 目录面板

- 从阅读器中可呼出目录面板（侧边抽屉或弹窗）
- 目录按章节序号排列
- 点击章节标题跳转至对应章节
- 当前阅读章节高亮显示
- 目录面板支持搜索章节（可选）

## 5. 非功能性需求

### 5.1 性能要求

- 首页加载时间 < 3 秒
- 章节内容加载时间 < 1 秒
- 翻页切换流畅，无明显卡顿

### 5.2 兼容性要求

- 支持主流浏览器：Chrome、Firefox、Safari、Edge
- 支持移动端浏览器自适应（响应式设计）

### 5.3 安全要求

- 用户密码使用 bcrypt 加密存储
- API 接口使用 JWT Token 鉴权
- 防止 SQL 注入、XSS 攻击
- 验证码接口限流防刷

### 5.4 数据存储

- 用户数据、小说数据、章节数据均存储于 MySQL
- 图片资源（头像、封面）存储于服务端本地或对象存储
